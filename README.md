# instagram-sraper
Boilerplate for a Finic project to scrape instagram

## Instructions

### Step 1
Create `input.json` and fill in the fields. For example:
```json
{
  "tags": ["marketing"],
  "num_results": 20
}
```

### Step 2
```json
poetry install

```

### Step 3
```json
poetry run query-to-username

```
